## YourWave — Бізнес профіль

### Назва та концепція
- **Назва:** YourWave
- **Слоган:** "Збери свою хвилю. Відкривай нову каву кожен день."
- **Концепція:** Coffee Discovery Experience — не просто кава, а відкриття нових смаків
- **Ключове відчуття:** Відкриття + Подарунок собі + Ритуал

### Профіль засновника
- Локація: Прага (вже там)
- Досвід кави: глибокий (знається на походженнях, обробці)
- Команда: соло
- Бюджет: €5,000–15,000
- Таймлайн: 6–12 місяців
- Аудиторія: старт з нуля

### Продуктова стратегія
- Формати: зерно/мелена + дріп-беги + поди + mix box (bundle builder)
- Bundle builder = core UX — клієнт сам складає свою хвилю
- Аудиторія: психографічна (coffee explorers), не демографічна

### Phase 1 стратегія (узгоджено)
- **Contract roasting** — без власного ростера на старті
- Знайти партнера-роастера в CZ (Chroast, Industra або інші)
- Власний ростер (Aillio Bullet R2, ~€3,700) — Phase 2 після перших продажів
- Старт з 3–5 SKU (2–3 походження, 2 формати)

### Продуктові лінії (потенціал)
- Wave Origins — ротаційна колекція по країнах
- Morning Wave — ранковий ритуал, підписка
- Explorer Wave — discovery box, щомісячна ротація
- Limited Wave — мікролоти, лімітовані дропи
- Guest Wave — куратори: барісти, музиканти, художники
