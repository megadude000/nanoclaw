    **2. Create `src/swarm-webhook-manager.ts`:**

    Export a Zod schema `SwarmIdentitySchema` = `z.object({ name: z.string().min(1), avatarURL: z.string().url() })` and `SwarmIdentitiesConfigSchema` = `z.array(SwarmIdentitySchema)` (per D-07).

    Export `loadSwarmIdentities(configPath?: string): SwarmIdentity[]` function following the same pattern as `webhook-router.ts`: `readFileSync` -> `JSON.parse` -> `SwarmIdentitiesConfigSchema.parse()`. If file doesn't exist, return empty array (graceful degradation). Config path defaults to `resolve(__dirname, '..', 'config', 'swarm-identities.json')`.

    Export `SwarmWebhookManager` class:
    - Constructor takes `SwarmIdentity[]` array, builds internal `Map<string, SwarmIdentity>` keyed by lowercase name (per D-08).
    - Private `cache = new Map<string, Webhook>()` keyed by `{channelId}:{identityName}` (per D-09).
    - `hasIdentity(senderName: string): boolean` -- checks if lowercase sender exists in identities map.
    - `async hydrateCache(channels: TextChannel[]): Promise<void>` -- for each channel, calls `channel.fetchWebhooks()`, filters by `wh.name?.startsWith('NanoClaw-')` AND `wh.token` truthy, maps to identity name by stripping `NanoClaw-` prefix and lowercasing, stores in cache (per D-10, D-03).
    - `async send(channel: TextChannel, text: string, senderName: string): Promise<boolean>`:
      1. Check `hasIdentity(senderName)` -- return false if unknown.
      2. Build cache key `{channel.id}:{senderName.toLowerCase()}`.
      3. Check cache for existing webhook.
      4. If not cached: call `channel.createWebhook({ name: 'NanoClaw-{senderName}', avatar: identity.avatarURL, reason: 'NanoClaw swarm identity: {senderName}' })` (per D-01, D-02). On failure, log with pino (per D-12) and return false (per D-11).
      5. Cache the webhook (per D-09).
      6. Chunk text using `chunkMessage(text)` from `discord-chunker.ts`.
      7. For each chunk, call `webhook.send({ content: chunk, username: identity.name, avatarURL: identity.avatarURL })` (per D-04, SWRM-01, SWRM-02).
      8. On send failure: log warning (per D-12), delete stale cache entry, return false.
      9. Return true on success.
    - `private webhookName(senderName: string): string` returns `NanoClaw-${senderName}`.
    - Use pino logger for all logging (per D-12).
