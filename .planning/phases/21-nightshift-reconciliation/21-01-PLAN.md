    Types to export:
    ```typescript
    interface StaleEntry { filePath: string; cortexLevel: string; daysSinceUpdate: number; ttlDays: number; }
    interface DiscoveredLink { source: string; target: string; score: number; }
    interface OrphanEntry { filePath: string; reason: string; }
    interface ReconciliationReport { staleEntries: StaleEntry[]; newLinks: DiscoveredLink[]; orphans: OrphanEntry[]; runAt: string; durationMs: number; }
    interface ReconciliationOptions { stalenessTTLs?: Record<string, number>; cosineThreshold?: number; maxLinksPerEntry?: number; }
    ```

    Functions:
    1. `checkStaleness(cortexDir: string, ttls?: Record<string, number>): StaleEntry[]`
       - Use globSync from node:fs to find all .md files in cortexDir
       - Parse each with parseCortexEntry(filePath, 'permissive')
       - Skip invalid entries (validation.valid === false)
       - Read `updated` or `last_updated` or `created` from frontmatter (NOT file mtime — Pitfall 2)
       - Compare days since update against STALENESS_TTLS[cortex_level]
       - Default TTLs from STALENESS_TTLS if not overridden

    2. `discoverCrossLinks(qdrant: QdrantClient, graphPath: string, threshold = 0.85, maxLinksPerEntry = 3): Promise<DiscoveredLink[]>`
       - Scroll all points from 'cortex-entries' collection with with_vector: true, with_payload: true
       - For each point, search with its vector at score_threshold = threshold, limit = maxLinksPerEntry + 1
       - Filter out self-matches (same file_path)
       - Filter out existing edges via hasEdge(graph, source, target, 'CROSS_LINK')
       - addEdge for each new link
       - saveGraph once at the end if any new edges added
       - Return array of { source, target, score }

    3. `findOrphans(cortexDir: string, graphPath: string): OrphanEntry[]`
       - Glob all .md files in cortexDir
       - Load graph and build index via buildIndex()
       - An entry is orphan if ALL true:
         a. No edges in graph (getNeighbors returns empty)
         b. Missing required frontmatter fields (cortex_level, confidence, domain, scope)
         c. Content body < 50 characters
       - Return array of { filePath, reason }

    4. `runReconciliation(cortexDir: string, graphPath: string, qdrant: QdrantClient, options?: ReconciliationOptions): Promise<ReconciliationReport>`
       - const start = Date.now()
       - Run checkStaleness, discoverCrossLinks (try/catch for Qdrant failure), findOrphans
       - Return { staleEntries, newLinks, orphans, runAt: new Date().toISOString(), durationMs: Date.now() - start }

    CRITICAL: Import STALENESS_TTLS from './types.js', parseCortexEntry from './parser.js', graph functions from './cortex-graph.js'. Use `import { globSync } from 'node:fs'` for file enumeration. Do NOT use node:fs/promises glob (it requires await and is async — globSync is simpler for this synchronous scan).

    Use pino logger from '../logger.js' for debug/info/warn logging.
