  <action>
    Create `scripts/bootstrap-multi-project.ts` implementing the same pattern as `scripts/bootstrap-cortex.ts` but sourcing from vault markdown files instead of TypeScript source.

    **Source material per project:**
    - YourWave: all .md files in `cortex/Areas/Projects/YourWave/` (YourWave.md + yw.*.md + Research/*.md)
    - ContentFactory: all .md files in `cortex/Areas/Projects/ContentFactory/` (ContentFactory.md + cf.*.md)
    - NightShift: all .md files in `cortex/Areas/Projects/NightShift/` (NightShift.md + nightshift.architecture.md + cron-registry.md)

    **generateProjectEntries(projectSlug, vaultDocs) function:**
    For each source markdown file, produce one Cortex entry:
    - Strip existing frontmatter (gray-matter parse), use body content as entry body
    - Assign cortex_level: files with 'architecture', 'spec', 'pipeline', 'cron' in name → L20; others → L10
    - Set project: projectSlug, domain: projectSlug, confidence: 'high'
    - Set scope: `${projectSlug} — ${filename without .md}`
    - Set type: 'bootstrap-extract', tags: [projectSlug, 'bootstrap', inferCategory(filename)]
    - Vault output path: `cortex/Areas/Projects/{PascalCase}/bootstrap/{filename}.md`
    - Content-hash skip: if entry already exists with matching source_hash, skip (reuse deterministicId pattern from embedder.ts)

    **inferCategory(filename):** 'platform' if filename includes 'platform'/'spec'/'architecture', 'ops' if 'ops'/'cron'/'legal', 'content' if 'atlas'/'pipeline'/'content', 'market' if 'market'/'branding', else 'core'.

    **Hub entry per project:** generate a `{Project}.md` hub file at vault root listing all bootstrapped entries (same as generateHubFile in bootstrap-cortex.ts), cortex_level L40.

    **CLI flags (identical to bootstrap-cortex.ts):**
    - `--dry-run`: print entries, no write, no embed
    - `--write-only`: write files, skip embedding
    - default: write + embed (requires Qdrant + OPENAI_API_KEY)
    - `--project yourwave|contentfactory|nightshift|all`: filter to one project (default: all)

    **Entry count target:** 10-20 entries per project (each source md file → one entry, plus hub). If a source file body < 50 chars after stripping frontmatter, skip it.

    **Imports:** reuse `embedEntry`, `createOpenAIClient` from `../src/cortex/embedder.js` and `createQdrantClient`, `checkQdrantHealth` from `../src/cortex/qdrant-client.js` — identical to bootstrap-cortex.ts.

    **Project slug → Pascal case mapping:**
    - yourwave → YourWave
    - contentfactory → ContentFactory
    - nightshift → NightShift
