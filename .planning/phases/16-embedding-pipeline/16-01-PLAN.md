    Create `src/cortex/embedder.ts` with:

    1. **Constants:**
       - `EMBEDDING_MODEL = 'text-embedding-3-small'`
       - `MIN_CONTENT_LENGTH = 50`

    2. **EmbedResult type:**
       ```typescript
       export interface EmbedResult {
         status: 'embedded' | 'skipped' | 'error';
         filePath: string;
         reason?: string;
       }
       ```

    3. **createOpenAIClient():** Per D-05, use readEnvFile(['OPENAI_API_KEY']) from src/env.ts. Throw Error with clear message if key missing. Return `new OpenAI({ apiKey })`.

    4. **deterministicId(filePath: string): string** -- Strip everything before 'cortex/' to get relative path. MD5 hash and format as UUID (xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx). Use node:crypto createHash('md5').

    5. **updateFrontmatter(filePath: string, updates: Record<string, string>): void** -- Read file with readFileSync, parse with gray-matter, Object.assign(data, updates), write back with matter.stringify(content, data). This writes source_hash and embedding_model back per D-03.

    6. **embedEntry(filePath, openai, qdrant, options?): Promise<EmbedResult>** -- The core pipeline per D-01/D-03:
       a. Call parseCortexEntry(filePath, 'permissive')
       b. If validation not valid, return error with validation errors
       c. If content.trim().length < MIN_CONTENT_LENGTH, return skipped 'content too short'
       d. If !options?.force AND frontmatter.source_hash === sourceHash, return skipped 'content unchanged'
       e. Call openai.embeddings.create({ model: EMBEDDING_MODEL, input: content })
       f. Extract vector from response.data[0].embedding
       g. Build payload: file_path, cortex_level, confidence, domain, scope, project (from frontmatter.project or validation.data.domain), status (from frontmatter.status or 'active'), source_hash, embedding_model
       h. Call qdrant.upsert(COLLECTION_NAME, { wait: true, points: [{ id: deterministicId(filePath), vector, payload }] })
       i. Call updateFrontmatter(filePath, { source_hash: sourceHash, embedding_model: EMBEDDING_MODEL })
       j. Return { status: 'embedded', filePath }
       k. Wrap in try/catch -- on error return { status: 'error', filePath, reason: error.message }
